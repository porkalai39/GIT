# GIT Merging

# Git Current branch
```
git branch
```
# Git list of branches
```
git branch -a
```

#To know the difference between the main and other branch
```
git diff new_branch
```
# Move to main branch
```
git checkout main
```

# Git Merge
```
git merge new_branch
```
# Git push to the main branch (reflect in github)
```
git push origin main
```
