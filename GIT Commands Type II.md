# GIT Commands Type II

# Initialize the Git 
```
git init
```
# Git Status
#View the Git status
```
git status
```
#Add the untracked files to the staging area
```
git add filename
(or)
git add .
```
#Commit the changes to remote directory
```
git commit -m "message"
```
#Check the current branch
```
git branch
```
